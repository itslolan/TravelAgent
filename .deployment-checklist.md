### Option 2: Manual Deployment
#### Python CAPTCHA Solver Service
- [ ] Web service created
- [ ] Root directory: `python-captcha-solver`
- [ ] Build command: `pip install -r requirements.txt`
- [ ] Start command: `gunicorn --bind 0.0.0.0:$PORT --workers 2 --timeout 120 captcha_solver:app`
- [ ] Environment variables set:
  - [ ] `GEMINI_API_KEY`
  - [ ] `FLASK_ENV=production`
- [ ] Health check path: `/health`
- [ ] Auto-deploy enabled
- [ ] Service deployed and URL copied

#### Node.js Backend Service
- [ ] Web service created
- [ ] Build command: `npm install && npm run build`
- [ ] Start command: `node server/index.js`
- [ ] Environment variables set:
  - [ ] `BROWSERBASE_API_KEY`
  - [ ] `BROWSERBASE_PROJECT_ID`
  - [ ] `GEMINI_API_KEY`
  - [ ] `NODE_ENV=production`
  - [ ] `USE_GEMINI_FOR_CAPTCHA=true`
  - [ ] `PYTHON_CAPTCHA_SOLVER_URL` (from Python service)
- [ ] Health check path: `/health`
- [ ] Auto-deploy enabled
- [ ] Service deployed successfully
